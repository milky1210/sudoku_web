# 数独アプリ - 機能仕様

## 実装済み機能

### モバイルファーストデザイン
- スマートフォン向けの縦型レイアウト
- PCブラウザでアクセスした場合、スマートフォンサイズ（最大420px幅）のビューポートに制限
- タッチ操作に最適化されたUI

### ゲーム機能
- 9x9の数独グリッド表示
- 自動パズル生成（難易度：40マス空白）
- 固定セル（問題）と入力可能セルの区別

### 入力方法
- **タップ入力**: セルをタップすると数字選択パネル（1-9）が表示され、選択可能
- **メモモード**: トグルボタンでメモモードに切り替え可能
  - メモモードON時: 複数の候補数字を右上に小さく記載
  - メモモードOFF時: 実際の答えを中央に大きく記載

### 検証機能
- リアルタイムエラーチェック（「チェック」ボタン）
- 行・列・3x3ブロック内の重複を検出
- エラーセルを赤くハイライト
- 完成時の正解メッセージ

### その他
- 新しいゲーム生成
- リセット機能（入力をクリア）

## 技術スタック
- Vue 3 (Composition API)
- TypeScript
- Vite
- CSS Grid Layout

## 開発サーバー起動方法
```bash
npm run dev -- --host 0.0.0.0
```

## 設計上の注意点
- モバイルデバイスでの操作性を最優先
- PC環境では中央にスマホサイズのコンテナを表示
- すべての操作はタップ/クリックで完結
